# JT809 项目结构

## 最终项目结构

```
iot-v-jt809/
├── pom.xml                                    # 父POM
├── docs/                                       # 文档目录
│   ├── 设计文档.md
│   └── 项目结构.md
│

│       │   ├── JT809Constant.java            # 协议常量

│       │

│       │   │
│       │   ├── upstream/                      # 上行消息（下级→上级）
│       │   │   ├── UpConnectReq.java         # 上行连接请求(0x1001)
│       │   │   ├── UpConnectResp.java        # 上行连接响应(0x1002)
│       │   │   ├── UpDisconnectReq.java      # 上行断开请求(0x1003)
│       │   │   ├── UpDisconnectResp.java     # 上行断开响应(0x1004)
│       │   │   ├── UpLinkTestReq.java        # 链路保持请求(0x1005)
│       │   │   ├── UpLinkTestResp.java       # 链路保持响应(0x1006)
│       │   │   └── vehicle/                  # 车辆相关上行消息
│       │   │       ├── VehicleLocationMsg.java         # 车辆定位(0x1200)
│       │   │       ├── VehicleAlarmMsg.java            # 车辆报警(0x1401)
│       │   │       └── VehicleRegisterMsg.java         # 车辆注册(0x1601)

│       │       ├── DownConnectReq.java       # 下行连接请求(0x9001)
│       │       ├── DownConnectResp.java      # 下行连接响应(0x9002)

│       │       ├── DownDisconnectResp.java   # 下行断开响应(0x9004)
│       │       ├── DownLinkTestReq.java      # 链路保持请求(0x9005)
│       │       ├── DownLinkTestResp.java     # 链路保持响应(0x9006)
│       │       └── vehicle/                  # 车辆相关下行消息
│       │           ├── VehicleLocationQueryReq.java    # 定位查询(0x9201)
│       │           └── VehicleControlReq.java          # 车辆控制(0x9301)
│       │

│       │   ├── JT809Encoder.java             # 消息编码器
│       │   ├── EscapeHandler.java            # 转义处理

│       │
│       ├── session/                           # 会话管理

│       │   ├── SessionManager.java           # 会话管理器
│       │   └── SessionListener.java          # 会话监听器
│       │
│       ├── util/                              # 工具类
```

## 各模块依赖说明

### jt809-core
- **职责**: 核心协议实现，不依赖Spring
- **依赖**: 
  - Netty (网络通信)
  - Lombok (代码简化)
  - SLF4J (日志门面)

### jt809-server
- **职责**: 服务端实现
- **依赖**:
  - jt809-core
  - Spring Context (可选，用于集成)

### jt809-client
- **职责**: 客户端实现
- **依赖**:
  - jt809-core
  - Spring Context (可选)

### jt809-handler
- **职责**: 消息处理器机制
- **依赖**:
  - jt809-core
  - Spring Context (用于Spring集成)

### jt809-spring-boot-starter
- **职责**: Spring Boot自动配置
- **依赖**:
  - jt809-core
  - jt809-server
  - jt809-client
  - jt809-handler
  - Spring Boot Autoconfigure

### jt809-example
- **职责**: 使用示例
- **依赖**:
  - jt809-spring-boot-starter

## 开发顺序建议

1. **第一阶段**: 核心基础
   - jt809-core
     - 消息模型定义
     - 编解码器实现
     - 工具类开发

2. **第二阶段**: 网络通信
   - jt809-server
   - jt809-client
   - 基本连接和消息收发

3. **第三阶段**: 处理器机制
   - jt809-handler
   - 处理器链实现
   - 内置处理器

4. **第四阶段**: Spring集成
   - jt809-spring-boot-starter
   - 自动配置
   - 配置属性

5. **第五阶段**: 示例和测试
   - jt809-example
   - 单元测试
   - 集成测试
